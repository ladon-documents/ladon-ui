# Ladon UI Bundler

## Getting started
```bash
npm i
```

## Bundle
This script bundles all needed micro frontends from `node_modules` into a `dist` folder and generates a named [zip file](zipper.mjs)

```bash
npm run bundle
```

## Cleanup
Deletes `dist` folder and any generated zip files
```bash
npm run cleanup
```
